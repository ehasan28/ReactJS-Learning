# ReactJS Learning

A personal React learning repository built with Vite. This folder contains small practice projects focused on building UI components, working with reusable React components, and handling form input with state.

## Projects

### 1. Card Project
Path: `01-Card-Project`

- A card-based UI that renders multiple job opening cards from array data
- Focuses on props, component reuse, and rendering lists with `map()`

### 2. UI Project
Path: `02-UI-Project`

- A multi-section interface built with reusable components
- Focuses on component structure, layout design, and passing structured data through props
- Uses Tailwind CSS alongside React and Vite

### 3. Form Handling
Path: `03-Form-Handling`

- A simple form example using React state
- Focuses on controlled inputs, submit handling, validation, and resetting form values

## Tech Stack

- React
- Vite
- JavaScript (ES Modules)
- CSS
- Tailwind CSS in `02-UI-Project`

## Getting Started

Clone the repository and move into any project folder:

```bash
cd 01-Card-Project
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
